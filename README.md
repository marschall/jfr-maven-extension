JFR Maven Extension
===================

This Maven extension generates JFR events for a Maven build.

Usage
-----


```xml
<build>
  <extensions>
    <extension>
      <groupId>com.github.marschall</groupId>
      <artifactId>jfr-maven-extension</artifactId>
      <version>0.1.0</version>
    </extension>
  </extensions>
</build>
```
